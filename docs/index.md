## How It Works

### Luminous Nix: Natural Language ‚Üí System Management

**The old way** (memorize syntax, read man pages, struggle):
```bash
sudo systemctl enable --now docker.service
sudo usermod -aG docker $USER
nix-env -iA nixpkgs.firefox
```

**The Luminous Nix way** (just say what you want):
```bash
ask-nix "enable docker and add me to the docker group"
ask-nix "install firefox"
ask-nix "what's using all my disk space?"
```

**Voice control:**
```
You: "Hey Nix, install vim and tmux"
Luminous Nix: "I'll install vim and tmux. Should I proceed?"
You: "Yes"
Luminous Nix: "Installation complete. Both packages are ready to use."
```

**Features:**
- ‚ú® Natural language understanding (works with any phrasing)
- üé§ Full voice interface with wake words
- üîí Safety checks and preview before execution
- üåê Works offline with local AI models
- üîì Open source, MIT licensed
