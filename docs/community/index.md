## ğŸ’¬ Communication Guidelines

### Asking Good Questions

**Good questions:**
- Include context (what you're trying to do)
- Show what you've tried
- Include error messages/logs
- Are in the right channel

**Example:**
> I'm trying to install Firefox using Luminous Nix on NixOS 23.11. I ran `ask-nix "install firefox"` but got error XYZ. I've checked the FAQ and searched Discord but didn't find this specific error. Full error log: [paste]

### Giving Good Answers

**Good answers:**
- Are patient and kind
- Explain the "why," not just the "how"
- Point to docs for further learning
- Welcome follow-up questions

**Example:**
> That error happens when the Nix daemon isn't running. Try `sudo systemctl start nix-daemon`. This daemon manages Nix operations in the background. Here's more info: [link to docs]
